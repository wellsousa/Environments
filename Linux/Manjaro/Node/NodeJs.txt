 
Para instalar o NodeJs no Arch/Manjaro:

//instala a versão mais recente do node
sudo pacman -S nodejs npm

/*
    Para instalar uma versão especifica do Node.js
    Primeiramente vamos fazer a instalação do nvm (node version manager - https://github.com/creationix/nvm) através do nosso gerenciador de pacotes yaourt:
*/
yay -S nvm


/*
    Depois vamos adicionar o nvm ao nosso shell, no meu caso o zshrc:
*/
echo 'source /usr/share/nvm/init-nvm.sh' >> ~/.zshrc
source .zshrc 

/*
    Por fim, podemos listar todas as versões do node com o comando nvm ls-remote e instalar a versão desejada:
*/
nvm install 10


//para remover o node junto com as dependencias nao usadas por outros pacotes
sudo pacman -Rs nodejs npm
